PORTFOLIO

RenewU

  • Who

    • Built for the user that wants to explore and practice meditation and mindfulness

  • What

    • The app guides users through a video exploration of meditation and minfulness and provides a place where all the tools needed to have a successful mediation session are readily available

  • Where

    • renew-u.netlify.com

  • Why

    • The majority of free apps promoting mindfulness and mediation are usually cluttered with ads or are hard to navigate - RenewU removes the junk to create a space that is calm and accessible

  • How

    • RenewU was built using the Mongo, Express, React, Node (MERN) stack, React Video player and React Music player were integrated into the app, responsive UI was created with CSS Grid

Foodie Phonetics

  • Who

    • Built for the user that wants to learn how to say their favorite foods in a new language

  • What

    • The app uses a spaced repetition algorithm, words the user gets correct show up with less frequency than those they get incorrect, to effectively help the user maximize the learning experience

  • Where

    • foodie-phonetics.netlify.com

  • Why

    • It's easy to find an app that teaches the fundamentals of a language. Foodie Phonetics is for those that specifically want to learn about the best part of any culture, the food!

  • How

    • Foodie Phonetics was built using the Mongo, Express, React, Node (MERN) stack, a doubly linked list data structure was also built on the backend to track the users progress

  • GitHub Repo

    • For a more detailed look at the code behind Foodie Phonetics, check out the GitHub Repo

Pley

  • Who

    • Built for the user that wants currated suggestions for their next vacation destination

  • What

    • The app has full functionality and a sleek UI for the traveler and ambassador to set trip details, request suggestions for an activity or location, and select from ambassador suggestions. "cards"

  • Where

    • pley.netlify.com

  • Why

    • Using traditional apps when traveling can often lead to being inundated with too many options (aka "choice paralysis)." Pley creates a more personalized concierge experience for traveling

  • How

    • Pley was built using the Mongo, Express, React, Node (MERN) stack, geospatial searching, React drag and drop, and the integration of the YELP API were also used

  • GitHub Repo

    • For a more detailed look at the code behind Pley, check out the GitHub Repo
